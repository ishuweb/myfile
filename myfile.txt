1.Use % as unit to reize the element relatively to the parent element.
2.use rem as unit to resize the element relatively to the root element [html]. It uses font-size as base of root element.
3.Use em as unit to resize the element relatively to its parent element. It uses parent elements font-size as base.
If use unit in other css property ten it will multiplies the both font sizes of parent element and current element and therfore given value also
ex.: .parent { fonr-size: 10px;} .child { font-size: 3em; padding: 3em; } then orignal padding apply will be 3*3*10=90px.
4.Use vw as unit to resize as relatively to the 1% of available viewport width.
5.Use vh as unit to resize as relatively to the 1% of available viewport height 
6.Use vmax as unit resize as relatively to the 1% of availabel viewport smaller dimension either width or height. Only work until width or heigt.
7.Use vmin as unit resize as relatively to the 1% of available viewport larger dimension either width or height. Only work until width or heigt.

NOTE : vh and vw as only relative to the available screen viewport size.
NOTE : % are the relative to the its parent element no matter of available viewport size.

What is resolution ---
The resolution is the XXX x YYY format numbers representd the amount of total pixels available on a display surface.

Is more resoltuion good or bad?
More resolution means more pixels on a display surface therfore more pixel means more clear and sharp image.

Physical pixels vs. logical pixels---
pixels mentioned in displays and cemeras are the logical as they can be in any of numbers in a specific area of surface of display.
pixels mentioned in graphics design and digital animationa or web design industry are the physical as they are fixed in form of pre-defined lenth 1px=0.26mm.

Viewport vs. resolution---
a device resolution tell about how many pixels is situated on the surface of the display.
But viewport is fixed as physical pixel and tell the exact size of the display.

Standard and Average viewport size of mobile devices is the 400px (max-width:400px)
Standard and Average viewport size of tablet devices is the 800px (max-width: 800px)
Standard and Average viewport size of desktop device is more than 800px (min-width: 800px)

========================= Firewall System =====================================
It is a software the installed on system to prevent unwanter access from Internet
Firewall maintain a list called 'access-control-list' 
Access-control-list has six parameters 
1.Permission 2.IP Address 3.Protocol 4.Destination 5.Port 6.Application
Firewall denies access to system if permission is denied to specific rules
Firewall let access to system if permission is allowed to specific rules
Firewall can be installed on single system called host-based firewall
Firewall can be single hardware that protect whole intranet infrastructure called network-based firewall
Firewall can be a part of anti-virus or router


Outbound rules are applied when we send request to internet
Inbound rules are appplies when we receiver request from internet
we can add, modify,view or delete rules as well
To know the IP address of any site execute 'ping domain.com' in cmd
